# Syncora Android client

Native Android and Android TV client for Syncora. The first milestone validates
TV navigation and local-network access through the server's `/health` endpoint.
Native low-latency H.264 and audio playback will follow once this foundation has
been tested on real hardware.

Open this directory as a project in Android Studio. The application supports
Android 6.0 (API 23) and newer and appears in both regular and Android TV launchers.
